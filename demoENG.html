⚡customSelect.js

Modern, accessible custom select for your projects

1. Simple Select

Just pass an array of strings:

// Simplest usage

new CustomSelect(['Apple', 'Banana', 'Orange'])

2. Select with Placeholder

Use an object for fine-tuning:

// Advanced usage

new CustomSelect({ options: ['New York', 'London', 'Tokyo'], placeholder: 'Choose city...' })

3. Working with Events

Track value changes:

// Subscribe to changes

select.on('change', (value) => { console.log('Selected:', value) document.getElementById('selectedValue').textContent = value })

4. Placement in Any Container

Selects can be placed anywhere:

// In element with ID

new CustomSelect(['Option A', 'Option B'], 'container1') // Directly in DOM element new CustomSelect(['Option X', 'Option Y'], document.getElementById('container2'))
