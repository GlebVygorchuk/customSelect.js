⚡customSelect.js

Современный, доступный кастомный селект для ваших проектов

1. Простой селект

Просто передайте массив строк:

// Простейшее использование

new CustomSelect(['Яблоко', 'Банан', 'Апельсин'])

2. Селект с плейсхолдером

Используйте объект для тонкой настройки:

// Продвинутое использование

new CustomSelect({ options: ['Москва', 'СПб', 'Казань'], placeholder: 'Выберите город...' })

3. Работа с событиями

Следите за изменениями значения:

// Подписка на изменения

select.on('change', (value) => { console.log('Выбрано:', value) document.getElementById('selectedValue').textContent = value })

4. Размещение в любом контейнере

Селекты можно размещать где угодно:

// В элементе с ID

new CustomSelect(['Опция A', 'Опция B'], 'container1') // Прямо в DOM-элементе new CustomSelect(['Опция X', 'Опция Y'], document.getElementById('container2'))
